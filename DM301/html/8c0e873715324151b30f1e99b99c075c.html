An essential part of the platform is the Learning Sequence Bar above every page. This bar allows you to move between the pages. It has three main features:

  • Previous Page Button: This button takes you to the previous page you are currently on.
  • Next Page Button: This button takes you to the next page you are currently on.
  • Content Pages: Every content page has a specific learning purpose and activity.

For example, to go to the next page, click on the fourth tab of the bar or click the Next button.
